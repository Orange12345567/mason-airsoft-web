
# Mason Airsoft Web App Setup

This is a simple static website that includes:
- Admin login (password only: Charcoal11!)
- Calendar to propose and approve/deny match dates
- Points system (10 pts per approved match)
- Shop with rewards redeemable using points

## How to Use

1. Unzip the project files
2. Open `index.html` in a browser (no server needed)
3. Click "Admin Login" and use password: Charcoal11!
4. Add dates, approve/deny them, and manage the shop

Enjoy!
